Import degli acquisti e scadenze, sono supportati solo i seguenti formati: .xls, .xlsx

NOTA: I dati degli articoli vengono sempre aggiunti.
Se l'utente non è presente viene inserito.
La prima riga del file viene considerata come intestazione, i nomi delle colonne (anche non in ordine) da utilizzare:
  • LINGUA, lingua di riferimento dell'utente, se non presente viene preso l'italiano
  • TIPOLOGIA, intra o extra UE
  • E-MAIL, e-mail dell'utente, campo obbligatorio
  • NOME, nome dell'utente
  • COGNOME, cognome dell'utente
  • TELEFONO, numero di telefono dell'utente
  • CODICE ARTICOLO, codice articolo (se non presente è obbligatorio specificare l'id)
  • ID ARTICOLO, id articolo (se non presente è obbligatorio specificare il codice)
  • NUMERO SERIALE, numero seriale articolo
  • DATA ACQUISTO, data di acquisto o di ultimo aggiornamento
